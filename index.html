How to Use

ALL RANDOMIZATION WILL BE DONE IN THE "RANDOMIZE" TAB.

To randomize bosses: Please select the regions for which bosses you would like to include in the pool. Weekly bosses are their own section and not included in any region. Then select the number of bosses you would like to randomize. If the number selected is larger than the pool, all bosses in the pool will be included in the output and no more. Finally click "randomize".

To randomize characters: Please go to the "Profiles" tab to either import a profile JSON or make a new profile. Then go to the "Randomize" tab to select P1 to P4 using the dropdown (dropdown options are the profiles in your "profiles" tab). If you would like to include banned characters in the pool, toggle bans to be off (bans default to being on). Then click "randomize".

"JACKPOT MODE" is a currently in development, by the request of friends. When active, there is a small chance of banned characters being selected.

Have fun!
